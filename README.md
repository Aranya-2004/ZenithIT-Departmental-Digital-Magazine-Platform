# ZenithIT

ZenithIT is the official departmental digital magazine platform developed for the Department of Information Technology.

The platform provides an engaging online reading experience for students, faculty members, and visitors through an interactive magazine interface, achievement showcases, event coverage, and departmental updates.

## Features

- Interactive Digital Magazine
- Flipbook-style Reading Experience
- Faculty Achievement Showcase
- Student Achievement Showcase
- Responsive Design
- Mobile-Friendly Interface
- Dynamic Content Rendering
- Archieve Section
